Запуск php з консолі в windows
Я незнаю де у вас знаходиться файл php.exe тому пропоную для експеременту зробити так
1)Гуглим "php для windows" або зразу переходимо http://windows.php.net/download/
2)Качаэмо перший лінк php7 "VC14 x86 Non Thread Safe (2016-Mar-30 21:40:40)" зіповський файл "http://windows.php.net/downloads/releases/php-7.0.5-nts-Win32-VC14-x86.zip"
3)Розпаковуэмо архів в c:/php
4)Правий клік на "Мой компьютер" -> Дополнительные параматры -> Переменные среды -> клікаємо на PATH і изменить 
5)Клікаємо в строку Значение переменной в кінці існуючої строки пишемо ; - щоб закрити попередній адрес і написати новий, повинно вийти так 
"старий запис;C:/php"
c:/php адресса до папки куди ми розпакували php
6)перезагружаэмо компьютер
7)Йдем в пуск -> в строку пошуку пишемо cmd.exe
8)в консолі пишемо "php -r phpinfo();"
фраг -r означаэ що php одразу виконаэ код в консолі
функція phpinfo(); виводить кучу інфи про сам php його модулі і середовище де він запущений
робили ми це просто для приколу щоб перевірити що все працює
9)запуск php файлу з консолі: в консоль пишемо "php c:/index.php" І жмем ентер 
с:/index.php адресса до вашого файлу php який ви хочете виконати
